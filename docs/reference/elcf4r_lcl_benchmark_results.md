# Low Carbon London benchmark results for shipped forecasting methods

Saved rolling-origin benchmark results for the shipped methods on a
fixed Low Carbon London cohort of households. The benchmark is based on
30-minute load curves and reports NMAE, NRMSE, sMAPE and MASE.

## Format

A data frame with the same benchmark-result schema as
`elcf4r_iflex_benchmark_results`.

## Source

Derived from the local LCL raw file with
`data-raw/elcf4r_lcl_artifacts.R`.
