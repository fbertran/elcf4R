# REFIT benchmark results for shipped forecasting methods

Saved rolling-origin benchmark results for the shipped methods on the
REFIT example cohort after resampling to 15-minute resolution. The
benchmark reports NMAE, NRMSE, sMAPE and MASE.

## Format

A data frame with the same benchmark-result schema as
`elcf4r_iflex_benchmark_results`.

## Source

Derived from the local REFIT raw files with
`data-raw/elcf4r_refit_artifacts.R`.
